GENERAL DESCRIPTION 
The 74HC164 is an 8-bit serial-in and parallel-out shift 
register which can accept a wide supply voltage range 
from 2.0V to 5.5V. 
This device provides gated serial inputs (DSA and DSB) 
and parallel data outputs (Q0 to Q7). DSA and DSB 
support serial data entry, where either input can allow 
data to enter through another input as an active high 
input. CP is a clock input. When the device is on 
low-to-high clock transition of the CP, data can be 
shifted. MR

level. The clamp diodes of inputs allow the use of 
current limiting resistors to connect inputs to the 
voltage exceeding supply voltage.

● +5.2mA/-5.2mA Output Current 
● CMOS Low Power Consumption  
● Gated Serial Data Inputs 
● Asynchronous Master Reset Input 
● -40℃ to +125℃ Operating Temperature Range
